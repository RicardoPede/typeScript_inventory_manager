# React + TypeScript + Vite

### Creación de entorno de trabajo
```bash	
npm create vite@latest
cd my-app
npm install
npm run dev
```

### Instalación de dependencias
```bash
npm install bootstrap
npm install react-router-dom
npm install @types/react-router-dom
npm install axios
npm install @types/axios
npm install react-hook-form
npm install @types/react-hook-form
npm install react-toastify
npm install @types/react-toastify
npm install react-icons
npm install @types/react-icons
npm install react-query
npm install @types/react-query
npm install react-query-devtools
npm install @types/react-query-devtools
npm install @mui/material
npm install @mui/icons-material
npm install @mui/system
npm install @mui/styled-engine-sc
npm install @mui/styled-engine-adapter
npm install @mui/styled-engine
npm install @mui/styled-engine-provider
